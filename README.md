# Exploración Vocacional ITS

Aplicación Web Progresiva (PWA) de Orientación Vocacional desarrollada para el Instituto Técnico Superior de Pereira.

## Descripción
Esta aplicación permite a los estudiantes descubrir su perfil vocacional a través de un test interactivo de 96 preguntas, evaluando su afinidad con 12 programas técnicos ofrecidos por la institución.

## Características
- **Interfaz Intuitiva:** Diseño limpio y fácil de usar en dispositivos móviles.
- **PWA:** Instalable como aplicación nativa en Android/iOS.
- **Resultados Inmediatos:** Cálculo automático de las áreas de mayor interés.
- **Integración con Google:** Envío automático de resultados a Google Sheets y notificación por correo electrónico.

## Tecnologías
- HTML5, CSS3, JavaScript (Vanilla)
- Google Apps Script (Backend)
